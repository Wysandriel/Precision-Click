# Precision Click｜精準點擊

一款使用 HTML、CSS、JavaScript 製作的簡單反應力網頁小遊戲。

## 遊戲簡介

Precision Click 是一款測試玩家反應速度、專注力與點擊準確度的小遊戲。

玩家需要在 30 秒內點擊畫面中隨機出現的白色目標。點擊正確目標可以獲得分數，反應越快分數越高；連續命中會增加連擊加成。若點錯位置，會扣分並中斷連擊。

## 功能特色

- 30 秒限時挑戰
- 隨機目標位置
- 分數系統
- 連擊加成
- 錯誤點擊扣分
- 平均反應時間統計
- 最高分紀錄
- 支援手機與電腦瀏覽器
- 可直接部署到 GitHub Pages

## 如何遊玩

直接用瀏覽器打開 `index.html` 即可開始遊戲。

## 上傳到 GitHub

1. 到 GitHub 建立一個新的 repository。
2. 將本專案的所有檔案上傳到 repository 根目錄。
3. 確認主頁檔案名稱是 `index.html`。
4. 進入 repository 的 Settings。
5. 找到 Pages。
6. Source 選擇 Deploy from a branch。
7. Branch 選擇 `main`，資料夾選擇 `/root`。
8. 儲存後等待 GitHub Pages 部署完成。
